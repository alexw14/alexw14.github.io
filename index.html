Projects

Sneak Peeks

Sneak Peeks is a crowdfunding platform for users to browse creative projects and contribute funding. Users can filter and search for projects by category. Users can also post projects and get funding from others. Users are able to see each project's current funding and number of backers.

Divide and Costco

Divide and Costco is an app that allows Costco shoppers to connect for savings. It allows two customers to connect shopping lists to split large quantity products. That way you reduce waste while also saving money on the items you need.

VRoam

VRoam is a concept e-commerce full-stack CRUD application built with Ruby on Rails. It includes a sign up/log in functionality and a shopping cart that stores items. I used Ruby on Rails, PostgreSQL, JavaScript and Bootstrap to build this application.

Pass the Minions

I built this client-side game that took inspirations from Pass the Pigs. It is designed to be played with 2 to 4 players. I developed this game with JavaScript, jQuery, HTML and CSS.
